Introduction

There is a lot of information out there on the internet about what coding is. It can relly be explained in a very simple way. It is way of giving instructions to computers or machines to perform specific tasks. It's like giving a set of directions to a robot to help it do what you want it todo.

Some examples of things that made possible by coding/programming are video games, websites and applications on a mobile phone.

Video games

Websites

Websites are created using a combination of HTML, CSS, and JavaScript. HTML is used to define the structure and content of a webpage, while CSS is used to control the look and feel of the page, and JavaScript is used to add interactivity and dynamic behavior to the page.

Applications on a phones

Mobile apps are created using programming languages such as Swift (for iOS apps) and Java (for Android apps). These apps can range from simple games to complex productivity tools and can be designed for use on a variety of different devices.

Social media platforms

Social media platforms, like Facebook and Twitter, are created using a combination of programming languages and technologies, including HTML, CSS, JavaScript, PHP, and databases like MySQL. These platforms allow users to share and connect with one another, and are powered by complex algorithms and data structures that are implemented in code.

E-commerce

E-commerce platforms, like Amazon and eBay, allow consumers to buy and sell products online. These platforms are created using a combination of programming languages and technologies, including HTML, CSS, JavaScript, PHP, and databases like MySQL, and are designed to handle large amounts of traffic and transactions.

HTML coding experience.

what is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating web pages.
  • HTML describes the structure of a web page.
  • HTML consists of a series of elements
  • HTML tells the browser how to display the content
  • HTML elemtnts lable pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML basic tags explained

  • The The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

What is an HTML element?

An HTML element is defined by a start tag(<), some content, and an end tag (>)

<tagname> Content goes here... </tagname>
The HTML element is everything from the start tag to the end tag:
<h1>My First Heading</h1>
<p>My first paragraph.</p>

Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!
